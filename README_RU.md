# SKR Airdrop

1. Скачай архив и распакуй.
2. В корне создай файл `.env` на основе `.env.example` и ВРУЧНУЮ вставь туда приватный ключ аирдроп-кошелька (base58 строка).
3. В терминале в папке проекта:
   npm install
   npm start
4. Открой в браузере http://localhost:3000 — увидишь сайт аирдропа.

Для деплоя на хостинг (Render/Railway и т.п.):
- залей файлы в репозиторий,
- укажи build: npm install, start: npm start,
- добавь переменную окружения AIRDROP_PRIVATE_KEY_BASE58 на самом хостинге.
